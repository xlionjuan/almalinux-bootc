# almalinux-bootc

```sh
sudo bootc switch ghcr.io/xlionjuan/almalinux-bootc:{VARIENT}
```

Or if the server is in internal

```sh
sudo bootc switch ghcr-oci-cache.xlion.dev/xlionjuan/almalinux-bootc:{VARIENT}
```

`{VARIENT}` currently have: `base`, `base-docker`, `misc`, `{VARIENT}-arm` for arm64 image.
